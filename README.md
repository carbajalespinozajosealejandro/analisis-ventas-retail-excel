# Análisis de Ventas Retail 2024

## Objetivo
Analizar 500 transacciones de ventas retail para identificar 
tendencias por región, rentabilidad por categoría y desempeño 
de canales y vendedores.

## Herramientas
- Microsoft Excel: Tablas dinámicas, SUMIFS, AVERAGEIFS, 
  INDEX/MATCH, XLOOKUP, IF anidados, Dashboard interactivo

## Preguntas respondidas
1. ¿Cuál es la tendencia de ventas por región a lo largo del año?
2. ¿Qué categorías de productos generan mayor rentabilidad?
3. ¿Cuál es el desempeño comparativo de los canales de venta?
4. ¿Qué vendedores tienen mejor performance?
5. ¿Cómo impactan los descuentos en el margen de ganancia?
6. ¿Cuáles son los productos estrella por categoría?

## Estructura del archivo
- Hoja 1: Documentación — metodología y preguntas de análisis
- Hoja 2: Datos — 500 transacciones limpias con métricas calculadas
- Hoja 3: Análisis — tablas resumen con fórmulas avanzadas
- Hoja 4: Dashboard — panel interactivo con KPIs y gráficos

## Dataset
500 transacciones · Enero–Diciembre 2024 · Ventas Retail
Regiones: Norte, Sur, Este, Oeste, Centro
Categorías: Electrónica, Deportes, Ropa, Hogar, Alimentos
